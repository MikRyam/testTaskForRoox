## Setup

```bash
git clone git@github.com:MikRyam/testTaskForRoox.git

npm install
# or
yarn install
```

## Getting Started

```bash
npm run start
# or
yarn start
```

## Other commands

```bash
# run ESLint
yarn lint

# run Prettier
yarn format
```

## Bundle application

```bash
# run build
yarn build
```
